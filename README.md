Projeto Final para a disciplina Tecnologias de Scripting e Automação

## Tema (idea do projeto)
<font size="4">**Sistema Automatizado de Gestão de Tarefas e Processamento de Ficheiros.**
<font size="3">
- Organização automática
- Ordenar por tipo (imagens, vídeos, texto)
- Ordenar por tamanho / data
- Mover para subpastas definidas pelo utilizador
- Converter formatos (ex.: png → jpg, mp4 → mp3)
- Comprimir / Descomprimir
- Monitorização;
- Gerar logs automáticos
